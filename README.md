
# Recreating a vintage Bulova "American Girl K" wristwatch 

After being popularized in the the show "The Queen's Gambit" this vinatge watch became virtually unobtainable on the second hand market.

My solution was to model the watch casing in CAD using pictures and video material as references and having the model cast in gold-plated bronze via 3D printed lost-mold casing.

This repository contains all CAD files - modeled utilizing both Fusion360 and Rhino - and makes them availiable to anyone who might wish to recreate the project.
